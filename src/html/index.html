Getting Started

CDN

You can consume the minified stored file at:

https://nakds.s3.eu-west-3.amazonaws.com/nakDS.min.css 
                
      <head>
        ...
        <link rel="stylesheet" href="https://nakds.s3.eu-west-3.amazonaws.com/nakDS.min.css" />
        ...
      </head>

NPM

Or install it via NPM

            
      npm i nakds
